# Telecom X – ETL e Análise de Evasão de Clientes

Este projeto tem como objetivo realizar a extração, tratamento e análise exploratória dos dados de clientes de uma empresa de telecomunicações, com foco na identificação de padrões relacionados à evasão de clientes (churn). O trabalho foi desenvolvido como parte do desafio Telecom X da formação em Data Science, que propõe a aplicação prática do processo de ETL utilizando Python.

Os dados utilizados no projeto foram obtidos a partir de uma API disponibilizada em um repositório no GitHub. A primeira etapa do projeto consistiu na extração desses dados utilizando Python. Após a coleta das informações, foi realizado o processo de transformação, que incluiu limpeza dos dados, tratamento de valores ausentes, verificação de inconsistências e organização das colunas para facilitar a análise.

Com os dados preparados, foi realizada uma análise exploratória com o objetivo de compreender melhor o comportamento dos clientes. Durante essa etapa foram geradas visualizações e análises estatísticas que ajudam a identificar padrões e possíveis fatores relacionados ao cancelamento de serviços. A análise também permitiu observar a distribuição de churn entre os clientes e avaliar relações entre diferentes variáveis do conjunto de dados.

O processo desenvolvido neste projeto demonstra a importância das etapas de ETL na análise de dados. A preparação adequada dos dados permite que análises mais profundas sejam realizadas posteriormente pela equipe de ciência de dados, contribuindo para a construção de modelos preditivos capazes de identificar clientes com maior probabilidade de evasão.

Este projeto foi desenvolvido utilizando Python e bibliotecas amplamente utilizadas na área de ciência de dados, como pandas para manipulação de dados, matplotlib e seaborn para visualização e requests para extração de dados da API. O notebook pode ser executado em ambientes como Jupyter Notebook ou Google Colab, desde que as bibliotecas necessárias estejam instaladas.
